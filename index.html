HEARTHSTONE

Hearthsone est un jeu de carte cree par l'editeur de jeu video Blizzard.
Le jeu est jouable sur :

  • PC
  • Android
  • IOS

Système de jeu

En début d'une partie, les joueurs choisissent entre les neuf héros de Warcraft (représentant eux-mêmes une des neuf classes de départ de World of Warcraft, avant les extensions) et un jeu de 30 cartes précédemment créé (un « deck »), pour disputer des batailles parmi l'un des quatre modes de jeu.

Les pesonnages jouables

Neuf personnages, issus de l'univers de Warcraft, sont présents dans le jeu. Ils possèdent chacun une compétence spécifique à leur classe, utilisable pour deux points de mana, et incarnent une des neuf classes de départ de World of Warcraft :

  • Uther dans le rôle de paladin
  • Jaina dans le rôle du mage
  • Anduin dans la rôle du prêtre
  • Rexxar dans le rôle du chasseur
  • Valeera dans le rôle du voleur
  • Guldan dans le rôle du démoniste
  • Malfurion dans le rôle du druide
  • Thrall dans le rôle du chaman
